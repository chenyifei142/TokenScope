更新类型：功能更新

### 新增

- 新增 Windows x64 安装包，支持选择安装目录、桌面快捷方式和开始菜单快捷方式。
- 新增统一数据目录解析；新安装默认使用 `安装目录\data`。
- 新增 `%APPDATA%\TokenSpider` 旧数据安全迁移，采用复制、验证、原子切换并保留旧目录。
- 自动更新改为下载并校验 `TokenMeter-Setup-vX.Y.Z-x64.exe`，随后沿用原目录静默覆盖安装。

### 优化

- PyInstaller 改为 onedir 构建，主程序固定为 `TokenMeter.exe`，更新器固定为 `TokenMeterUpdater.exe`。
- 覆盖安装和默认卸载均保留 `data`，更新后快捷方式继续指向原安装位置。
- GitHub Actions 依次执行测试、onedir 构建、Inno Setup 构建、smoke test 和 SHA256 校验文件生成。
- 更新安装、数据迁移、自动更新和卸载文档，并保持 `TokenMeter/`、`TokenSpider/`、`TokenScope/` 凭据目标兼容。

### 安全与兼容

- 迁移失败时继续使用旧数据目录，不影响程序启动，也不会移动、清空或删除旧目录。
- 配置文件和日志不保存 Cookie、Bearer Token 或 API Key，敏感信息继续使用 Windows Credential Manager。
- 安装包更新启动失败时保留当前可用版本，并撤销待清理状态。
