更新类型：功能新增

### 新增

- 新增「今日分时」Token 活动视图：以两次成功刷新间的当日三类 Token 累计差额进行分钟级均摊估算，并明确标注估算口径。
- 分时图支持缓存命中输入、未命中输入和输出三层堆叠展示、总量轮廓、图例开关、分钟悬停明细及全天缩放导航。
- 新增应用数据目录设置；重启后可安全迁移配置、历史数据库、日志、更新缓存和专用浏览器会话，迁移失败时保留原目录。

### 兼容性

- 分时缓存仅保留各提供商当天数据，不影响既有 `daily_usage`、年度活动和历史账单。
- MiMo 按 `Asia/Shanghai` 划分估算日期；DeepSeek 按运行设备本地时区划分。

### 验证

- 已覆盖分钟差额分摊、跨日清理、数据目录迁移、Cookie 专用目录及界面交互测试。
